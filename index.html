Objectives

This is process document explaining the worflow of virtual machine provisioning in azure infrastructue with end to end automation includes below list of tasks

  • SNOW Catalog Request Submittion
  • Creating Virtual Machine
  • Perform Postbuild Activity
Synopsis:
  • This is the process to provison virtual machine in azure (Windows/Linux operating system).
  • This automation workflow will perform post build activities and validate the virtual machine.
  • End user will submit service request with all the required virtual machine specifications.
  • Once the reqeuest is approved in SNOW,the catalog will trigger the Azure pipeline automatically.
  • The pipeline will then executes the task in one by one.
  • Logic App is integrated with SNOW to perform ticket Creation/Updation/Closure.
  • Engineer to update the raw github url with token in keyvault.
  • All credentials are stored in keyvault.
Input Parameters:
Parameter Type Description Required
subscription String Provide the subscription id Yes
resource_group String Provide the resource group name Yes
region String Provide the reagion Yes
environment String Provide the environment name DEV, NON-PROD, or PPROD Yes
managed String Choose yes or no to define it is managed by client Yes
